## Known Frameworks

```
GET /api/v1/harnesses/frameworks
```

No authentication required. Returns the canonical list of known frameworks and suggested taxonomy values for harness descriptors.

**Response:**

```json
{
  "ok": true,
  "data": {
    "frameworks": [
      {
        "id": "claude-code",
        "name": "Claude Code",
        "category": "cli",
        "url": "https://github.com/anthropics/claude-code",
        "defaultTools": ["bash", "read", "write", "edit", "grep", "glob"],
        "description": "Anthropic's agentic CLI for software engineering."
      }
    ],
    "suggested_loop_types": ["single-agent", "multi-agent", "swarm", "human-in-the-loop"],
    "suggested_context_strategies": ["progressive-disclosure", "static", "dynamic", "rag"],
    "suggested_error_strategies": ["model-driven", "code-driven", "linter-gated", "retry-loop"],
    "canonical_tools": ["bash", "read", "write", "edit", "grep", "glob"]
  }
}
```

All taxonomy values are suggestions — any string is accepted. The taxonomy evolves with usage.
